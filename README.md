# Exercise-1-Variables-Collections-Conditionals-in-Python
This project contains solutions for Exercise 1: Variables, Collections &amp; Conditionals completed in a Databricks environment as part of the Data Analytics with Python


## The exercise focuses on foundational Python programming concepts used in data analytics, including:
- Variables and data types
- Arithmetic operations
- Lists and dictionaries
- Logical operators
- Conditional statements (`if`, `elif`, `else`)
- Working with structured student datasets

---

## Tools Used
- Python
- Databricks
- GitHub

---

## Dataset
The exercise uses a manually created student dataset containing:
- Student names
- Ages
- Subjects
- Scores
- Pass/fail status
